AFRICAN SOIL PROPERTY PREDICTION¶

In this project, I build models to predict physical and chemical properties of soil using spectral measurements. The African Soil Property Prediction Dataset was obtained from kaggle website.

There are 5 target variables for prediction. These are:

  • Ca - Mehlich-3 extractable Calcium
  • P - Mehlich-3 extractable Phosphorus
  • SOC - Soil organic carbon
  • pH - pH values
  • Sand - Sand content

The dataset is comprised of 1157 observations and 3600 columns. In addition to the 5 columns containing the targets, the other columns contain mid-infrared absorbance measurements of soil samples, and some spatial measurements obtained from remote sensing data sources. A brief description of the variables are provided below:

  • PIDN - unique soil sample identifier
  • m7497.96 - m599.76 - There are 3,578 mid-infrared absorbance measurements. For example, the "m7497.96" column is the absorbance at wavenumber 7497.96 cm-1.
  • Depth - Depth of the soil sample (2 categories: "Topsoil", "Subsoil")
  • BSA - average long-term Black Sky Albedo measurements from MODIS satellite images (BSAN = near-infrared, BSAS = shortwave, BSAV = visible)
  • CTI - compound topographic index calculated from Shuttle Radar Topography Mission elevation data
  • ELEV - Shuttle Radar Topography Mission elevation data
  • EVI - average long-term Enhanced Vegetation Index from MODIS satellite images.
  • LST - average long-term Land Surface Temperatures from MODIS satellite images (LSTD = day time temperature, LSTN = night time temperature)
  • Ref - average long-term Reflectance measurements from MODIS satellite images (Ref1 = blue, Ref2 = red, Ref3 = near-infrared, Ref7 = mid-infrared)
  • Reli - topographic Relief calculated from Shuttle Radar Topography mission elevation data TMAP & TMFI: average long-term Tropical Rainfall Monitoring Mission data (TMAP = mean annual precipitation, TMFI = modified Fournier index)

All the variables in this dataset are of data type float, except for Depth which is a categorical variable.

Exploring the correlation of soil depth with each of the targets¶

Boxplots have been used to show the distribution of the target values for topsoil and subsoil. As shown in the boxplots, the distribution of each target value appears to be similar regardless of soil depth. T-test were performed to validate this observation. The table of p-values below shows that, at a 95% confidence level, we fail to reject the null that the mean target values of Ca, pH, P differ across the soil depth. At a 95% confidence level, we reject the null in favor of the alternative that there is significant difference between the mean target values of SOC and Sand across the soil depth.

As a result of the t-tests, the variable 'Depth' will be ignored in the prediction models for Ca and pH, but will be considered for SOC and Sand. Since the p-value from the t-test for P was 0.07 which is only slightly greater than 0.05, Depth will still be considered when building the prediction model.

Feature Selection¶

I used SelectKBest function, with a f-regression scoring to select the k best wavelength mid-infrared measurements for the different targets. The value of k was dependent the number required to build the best predictive model for each target. I made use of 600 wavelength measurements for Ca, SOC and Sand, and 1000 for pH.

Then I defined the X variables for each target. The X variables for targets Ca, and pH only consisted of mid-infrared measurements. For targets SOC and Sand, in addition to mid-infrared measurements I included soil depth as predictors, since they both had significant correlation with soil depth as discussed earlier.

When building the models, I found no value in adding any of the spatial predictors as there did not improve the models.

## Ridge Regression for Ca

RigR_Ca = Ridge(alpha=0.01, solver='auto', random_state=None)
RigR_Ca.fit(training_Ca_X, training_Ca_y)

## Learning Curve
def plot_curve_RigR_Ca():
    # instantiate
    RigR_Ca = Ridge(alpha=0.01, solver='auto', random_state=None)

    # fit
    X = data_Ca_X
    y = data_Ca_y
    RigR_Ca.fit(X, y)
    

    train_sizes, train_err, val_err = learning_curve(RigR_Ca, X, y, n_jobs=1, cv=100, train_sizes=np.linspace(.1, 1.0, 5), verbose=0,
                                                     scoring = "neg_mean_squared_error")
    
    train_err = abs(train_err)**0.5
    val_err = abs(val_err)**0.5

    train_err_mean = np.mean(train_err, axis=1)
    train_err_std = np.std(train_err, axis=1)
    val_err_mean = np.mean(val_err, axis=1)
    val_err_std = np.std(val_err, axis=1)
    
    plt.figure()
    plt.title("Ridge Regression Model for Ca")
   
    # plot the std deviation as a transparent range at each training set size
    plt.fill_between(train_sizes, train_err_mean - train_err_std, train_err_mean + train_err_std, alpha=0.1, color="r")
    plt.fill_between(train_sizes, val_err_mean - val_err_std, val_err_mean + val_err_std, alpha=0.1, color="g")
    
    # plot the average training and test score lines at each training set size
    plt.plot(train_sizes, train_err_mean, 'o-', color="r", label="Training score")
    plt.plot(train_sizes, val_err_mean, 'o-', color="g", label="Cross-validation score")
    
    plt.legend(loc='lower right')
    plt.xlabel("Training examples")
    plt.ylabel("Root Mean Squared Error")
    
    # box-like grid
    plt.grid()
    # sizes the window for readability and displays the plot
    # shows error from 0 to 1.1
    plt.ylim(-.1,1.1)
    plt.show()

Prediction Model for P¶

After several failed attempts to build a prediction model for P using the available predictors, I concluded that no good model can be built. This means that the available predictors (i.e. infrared spectra, and spatial measurements) have insufficient correlation with P. In the exploratory analysis, it was observed that P had no correlation with any of the other targets. This supports the conclusion that a different group of predictors will be required to build a prediction model for P.

Model Selection¶

When comparing the performance of each model on the different targets, I made the model choice for each target on the conditions of lower RMSE and better estimation of out-of-sample error. Below are the model selections for each of the targets:

  • For Ca - Ridge Regression model
  • For pH - SVR Model
  • For SOC - Ridge Regression Model
  • For Sand - Ridge Regression Model
